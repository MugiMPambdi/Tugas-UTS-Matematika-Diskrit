Penjelasan:
	1.	Fungsi is_even(n) memeriksa apakah bilangan n genap (n % 2 == 0).
	2.	Kita periksa untuk beberapa nilai n dari -20 sampai 20.
	3.	Program menampilkan hasil untuk semua bilangan genap, lalu memeriksa apakah n² juga genap.
  Dari hasil tersebut, setiap n genap menghasilkan n² yang juga genap, sehingga pernyataan terbukti benar.
